# 📚 Learn-AI

## 📌 Overview

Learn-AI is an AI-powered micro-learning and adaptive quiz platform developed using **Flutter** and **Dart**. The application provides personalized learning experiences through AI-powered recommendations, adaptive quizzes, progress tracking, and interactive learning modules.

The project aims to make learning smarter, more engaging, and personalized by leveraging Artificial Intelligence to improve students' learning outcomes and overall educational experience.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Tharshini-K/Learn-AI.git
```

### Navigate to Project

```bash
cd Learn-AI
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```
